Uses the static version as the basis for a force-directed graph, using an aspect of the Voronoi map (the Delaunay triangulation) to build links. The graph moves quickly away from the initial Hilbert mapping, but there are more links between nodes with near Dewey numbers, so they cluster together.

The same visualisation with 200 random values generates a uniform (but still jumbled) mesh.
